Тестовое задание для Selectel (Frontend)
Что сделано:
Мини-приложение на Angular 19 из двух страниц:

1. Главная страница — приветствие и ссылка на страницу с меню
2. Страница с меню — чекбоксы, сгруппированные по типам. В шапке отображается:
   - Выбранный тип меню
   - Количество выбранных пунктов
   - Сумма их значений

Стек
- Angular 19+
- TypeScript
- HTML/CSS
- standalone компоненты
- новый синтаксис @for

Как запустить:
npm install --legacy-peer-deps
npm start
